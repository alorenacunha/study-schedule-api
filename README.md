# Study Schedule

This project is a SPA Web Application developed in ReactJs to build a dashboard and manager of the study's schedule.

## Usage

### Deployment

Install dependencies with:

```
npm install
```

and then deploy with:

```
serverless deploy
```

### Local development

```bash
serverless offline
```
